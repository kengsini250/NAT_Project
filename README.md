## 使用方法
    1.建立ssh连接环境（需要设置免密码登录）
    2.在/home/用户/ 的地方新建一个http文件夹
    3.配置apache环境。默认文件夹改为http
    4.File->New->输入主机和ip
    
Setting 根据需求可以改变下载保存路径，下载路径默认是users的downloads文件夹
